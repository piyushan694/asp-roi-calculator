# Assets

Drop your Alexa+ brand logo here to have it appear in the hero of the ROI app.

Supported filenames (first match wins):
- `alexa-plus-logo.png` (preferred)
- `alexa-plus-logo.jpg`
- `alexa-plus-logo.svg`
- `alexa+.png`

The app looks for these on startup. If none are present, a text fallback is shown.
